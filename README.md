**Данные о населении**

Имеются данные о населении городов по всему миру. Для каждого города указана страна.
Для каждого города указано общая численность населения.
Кроме того для многих городов приведено как мужское, так и женское население.

Необходимо реализовать job, рассчитывающий:
 - население стран
 - для каждой страны:
    * количество городов-миллионников
    * топ 5 самых крупных городов
    * соотношение мужского и женского населения

Результаты должны сохраняться в базу данных MongoDB.
